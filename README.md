# Central VTC Perpignan

Site vitrine statique prêt à mettre sur GitHub Pages.

## Fichiers

- `index.html` : structure du site
- `style.css` : design responsive desktop/mobile
- `script.js` : menu mobile

## Mise en ligne GitHub Pages

1. Crée un dépôt GitHub, par exemple `central-vtc-perpignan`.
2. Ajoute les fichiers à la racine du dépôt.
3. Va dans **Settings > Pages**.
4. Choisis **Deploy from a branch**, puis branche `main`, dossier `/root`.
5. Clique sur **Save**.
